## 1. 产品概述

像素植物温室是一款像素风格的轻养成解压应用，用户通过完成日常习惯任务（喝水、拉伸、阅读等）来浇灌和培育虚拟像素植物，享受从种子到绽放的全过程陪伴感。面向学生、居家办公人群和解压用户，提供无社交压力、无复杂任务系统的纯粹治愈体验。

- 核心价值：将日常微习惯转化为可视化养成反馈，让用户在低负担中获得持续的成就感和陪伴感
- 市场定位：区别于重度社交养成游戏，以"独处治愈 + 习惯养成"为差异化方向

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览温室、查看图鉴预览 |
| 种植者 | 本地存储自动创建 | 种植植物、完成习惯任务、布置环境、收集图鉴 |

### 2.2 功能模块
1. **温室主页面**：展示当前所有植物的像素温室场景、环境装饰、天气效果
2. **种植详情页**：单株植物的详细生长状态、养护记录、习惯浇水面板
3. **习惯面板页**：今日待完成习惯列表、完成进度、习惯映射规则
4. **图鉴收集页**：已解锁/未解锁植物图鉴、稀有度展示、收集进度

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

用户进入温室 → 查看今日待完成习惯 → 选择习惯任务并打卡完成 → 习惯转化为浇水动作 → 植物获得生长值 → 达到阈值后植物生长阶段升级 → 解锁新图鉴 → 用图鉴奖励装饰温室环境

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：温暖绿色系（#4A7C59 草木绿）搭配柔米色背景（#F5F0E8）
- **辅助色**：土壤棕（#8B6F47）、天空蓝（#87CEEB）、日落橙（#E8945A）
- **按钮风格**：像素风格圆角按钮，带有3D像素凸起效果和按压动画
- **字体**：中文使用"ZCOOL QingKe HuangYou"（站酷庆科黄油体）作为像素风格标题字体，正文使用"Noto Sans SC"
- **布局风格**：居中大画布温室场景 + 底部工具栏 + 侧边抽屉面板
- **图标风格**：16x16 / 32x32 像素风图标，单色或低饱和度配色
- **整体氛围**：温馨、安静、治愈，像素风窗台小世界

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 温室主页面 | 温室场景 | 像素画风格窗台，CSS像素边框，缓慢摇摆的植物动画，天气粒子效果（雨滴/雪花/阳光光斑），背景音乐切换按钮 |
| 温室主页面 | 环境切换栏 | 底部像素风图标按钮组，选中态带发光边框，hover放大微动画 |
| 温室主页面 | 快捷操作 | 左上角签到按钮（像素日历图标），右下角"+"种植按钮 |
| 温室主页面 | 植物列表缩略 | 右侧抽屉，半透明毛玻璃背景，植物像素缩略图卡片，点击滑出详情 |
| 种植详情页 | 生长可视化 | 居中大尺寸植物像素动画（CSS帧动画），底部渐变进度条，阶段名称标签 |
| 种植详情页 | 习惯浇水面板 | 习惯卡片网格，完成态打勾+绿色发光，未完成态虚线边框 |
| 种植详情页 | 养护日志 | 垂直时间线，像素风格图标节点，日期+动作描述 |
| 习惯面板页 | 今日习惯 | 习惯任务卡片列表，每个卡片包含像素图标+习惯名+完成按钮，完成触发水滴飘落动画 |
| 习惯面板页 | 连续记录 | 顶部连续天数计数器，像素火焰图标，日历热力图 |
| 图鉴收集页 | 图鉴网格 | 3列网格布局，稀有度边框颜色（普通绿/稀有蓝/史诗紫/传说金），未解锁灰色剪影 |
| 图鉴收集页 | 植物详情 | 居中放大像素插画，底部文字介绍，解锁条件提示 |

### 4.3 响应式
- 桌面优先设计，最小宽度1024px完整体验
- 移动端自适应：温室场景缩放，侧边抽屉改为底部弹出面板，图鉴网格2列
- 触摸优化：按钮最小点击区域44x44px像素对齐，滑动切换环境

### 4.4 3D场景指导
- 不适用3D场景，采用2D像素艺术风格，通过CSS动画和Canvas实现像素风动态效果
